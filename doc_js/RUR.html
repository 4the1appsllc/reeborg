add_goal_object(specific_object, x, y, nb)

This function sets a specified quantity of a given object as a goal at a certain location. By "object" we mean a type of object that can be taken or put down by Reeborg.

Cette fonction spécifie la quantité d'un certain type d'objet qui doit être mis comme but à un endroit donné. Par "objet", on entend ici un objet qui peut être transporté ou déposé par Reeborg.

add_new_home_tile(name, url, info)

This function makes it possible to add a new image to use as a home tile. If the name of an existing home tile is specified again, the image for that object is replaced; we suggest you do not do this.

Cette fonction permet l'ajout de nouvelles images à utiliser comme but à atteindre pour le robot. Si le nom d'un but existant est spécifié, l'image pour ce but est remplacée par la nouvelle image; nous vous suggérons de ne pas faire ceci.

Parameters:
Name Type Description
name string

The name to be used for this tile
Le nom à utiliser pour cet objet.
url string

URL where the image can be found.
URL où l'image peut être trouvée
info string

A sentence to be displayed when the user queries information about the world; something like "goal: Reeborg can detect this tile using at_goal()."
Une phrase décrivant ce but lorsqu l'usager consulte la description du monde; quelque chosen comme "but: Reeborg peut détecter ceci en utilisant au_but()".

add_new_object_type(specific_object, url, url_goal)

This function makes it possible to add new objects. If the name of an existing object is specified again, the images for that object are replaced. Two images must be provided: one for the object itself, and another when this object is specified as a goal. N.B. existing names are the English ones.

Cette fonction permet l'ajout de nouveaux objets. Si le nom d'un objet existant est spécifié, les images pour cet objet seront remplacées par les nouvelles images fournies. N.B. les noms d'objets par défaut sont des noms anglais (par exemple "token" plutôt que "jeton").
